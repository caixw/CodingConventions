Git常用配置项
============


设置全局变量。
```shell
git config --global user.name caiw
```

设置当前项目变量。
```shell
git config --local user.name caixw
```

一些惯用的git设置。
```shell
user.email=xnotepad@gmail.com
user.name=caixw
push.default=simple
color.ui=true
core.editor=vim
core.filemode=false     # 忽略文件权限变更
core.ignorecase=false   # 不忽略文件名大小写变更
```
